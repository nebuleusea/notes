
# 切分窗口
|命令|用途|
|:-|-:|
|\<C-w> s|水平切分当前窗口，新窗口仍显示当前缓冲区|
|\<C-w> v|垂直切分当前窗口，新窗口仍显示当前缓冲区|

# 窗口间切换
|命令|用途|
|:-|-:|
|\<C-w> w|在窗口间循环切换|
|\<C-w> h|切换到左边的窗口|
|\<C-w> j|切换到下边的窗口|
|\<C-w> k|切换到上边的窗口|
|\<C-w> l|切换到右边的窗口|

# 关闭窗口
|Ex命令|普通模式命令|用途|
|:-|-|-:|
|:clo[se]|\<C-w> c|关闭活动窗口|
|:on[ly]|\<C-w> o|只保留活动窗口，关闭其他所有窗口|
