### 常用工具的命令（搜索栏，CMD）
```
calc #计算器
mspaint #画图
gpedit #编辑组策略
regedit #注册表编辑器
```

### ping
```cmd
ping url
ping ip
```

### ipconfig
```cmd
ipconfig
ipconfig /flushdns #刷新DNS
```

### netstat
```cmd
netstat
netstat -an #查看电脑网络端口使用情况
netstat -aon|findstr "端口号"
```

### taskkill
```cmd
taskkill /pid 进程号 -t -f #结束某个PID进程
```

### shutdown
```cmd
shutdown -s #立即关机
shutdown -s -t 100 #100秒后关机
shutdown -a #取消计划
shutdown -r #重启
```
